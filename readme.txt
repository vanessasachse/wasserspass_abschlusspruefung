Da die Browser scheinbar meine wellenförmigen svgs blockieren, wenn der Pfad im Browser file:/// ist, müsste die Seite, um diese angezeigt zu bekommen, auf einem lokalen Webserver gestartet werden.
Dafür reicht es schon, wenn z.B die Live Server Extension diverser Editor genutzt wird (oder eben über xampp, etc.)
Dafür also bitte die Seite auf einem lokalen Server öffnen. 

Die Seite ist wie gewohnt über die index.html zu starten. 
in dem Ordner assets befinden sich die Bilder, fonts, etc.
